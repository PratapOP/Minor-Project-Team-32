# 🧠 StressIntel PRO: Hybrid AI Stress Intelligence System

## Project Overview
**Minor Project Team 32**

StressIntel PRO is a research-grade, multi-modal stress assessment platform that combines machine learning, Explainable AI (XAI), and Large Language Model (LLM) reasoning to provide high-fidelity stress intelligence. This system is designed for institutional use to monitor student/employee well-being with transparent, actionable insights.

## ✨ Research Highlights
- **Multi-Modal Synthesis**: Combines Behavioral Surveys, Facial Biomarkers (Webcam), and Semantic Sentiment Analysis (Journaling).
- **Explainable AI (XAI)**: Utilizes **Kernel SHAP** to provide waterfall visualizations of stress contributors.
- **Cognitive Reasoning**: Integrates LLM-based feedback to provide personalized "Stress Mitigation Roadmaps."
- **Temporal Analysis**: Provides longitudinal tracking of stress trajectories for research validation.

## 🚀 Key Features
1. **Adaptive Assessment**: Dynamic survey with real-time sentiment adjustment.
2. **Mitigation Roadmap**: AI-generated lifestyle interventions based on individual triggers.
3. **Temporal Trends**: 30-day stress trajectory visualization.
4. **Research Lab**: Academic dashboard with Confusion Matrices, ROC Curves, and Correlation Heatmaps.

## 🛠️ Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Launch the application:
   ```bash
   python main.py
   ```

## 📊 Methodology
- **Model**: Extreme Gradient Boosting (XGBoost) Ensemble.
- **XAI**: SHAP (Shapley Additive Explanations) for local and global interpretability.
- **NLP**: Semantic Tone Detection using TextBlob.
- **UI**: Streamlit with Custom CSS injection and Glassmorphism components.
